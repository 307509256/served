## Installation

### Requirements

* [Boost 1.56](http://www.boost.org/)
* [Ragel](http://www.complang.org/ragel/)

### Building

```bash
$ git clone git@github.com:datasift/served.git
$ ./autogen.sh
$ ./configure && make && make check && make install
```
